PHP

PHP is an recursive acronym for PHP:Hypertext Preprocessor
where PHP here stands for Personal Home Page.
PHP is a widely-used, open source scripting language which is executed on the server and free to download and use

What does php need to run?

  • PHP need a platform i.e Unix/Windows/Mac.
  • PHP need a server(Apache).
  • PHP need wide range of database (mysql,MongoDB..).
  • And you will need a text editor for yourself.

Why PHP?

  • PHP is server side scripting language used for making dynamic websites.
  • PHP can do literally any thing.
  • PHP is easy to learn. One can easy learn php within a day and start coding.
